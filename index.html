Multiplayer Survival Game built with Unity and Photon Fusion

About the Game

Extreme World is an open-world multiplayer survival game focused on exploration, building, and player interaction. The project was developed using a networked architecture powered by Photon Fusion, ensuring smooth and scalable multiplayer gameplay.

Technical Overview

Engine: Unity

Networking: Photon Fusion

Architecture: Hybrid Multiplayer Architecture (Client-Server & Host-Client)

Session Types: Dedicated Servers and Player-Hosted Sessions

Networking Model: Authoritative Server with Client Prediction

Focus: Low latency, synchronization accuracy, and anti-wallhack system
